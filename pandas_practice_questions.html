Q1
How do you read a CSV file directly from a URL into a pandas DataFrame?
import pandas as pd
URL = "https://example.com/data.csv"
df = pd.read_csv(URL)
Q2
After reading a CSV from a URL, how do you skip specific rows that contain the phrase "this row is useless"?
import requests
lines = requests.get(URL).text.splitlines()
skip = [i for i, l in enumerate(lines) if "this row is useless" in l.lower()]
df = pd.read_csv(URL, skiprows=skip)

enumerate() gives (index, line) pairs; skiprows accepts a list of 0-based row indices to skip before parsing.

enumerate(iterable) returns tuples of (index, value):

Q4
How do you drop rows that have more than 1 NA value (i.e. keep rows with at most 1 NA)?
df.dropna(thresh=len(df.columns) - 1, inplace=True)
# thresh = minimum number of non-NA values required to keep a row
# len(df.columns) - 1  means at most 1 NA is allowed
Q5
How do you drop columns where ALL values are NA?
df.dropna(axis=1, how='all', inplace=True)
# axis=1  → operate on columns (axis=0 is rows)
# how='all' → only drop if every value is NA
# how='any' → drop if at least one value is NA
Q6
How do you fill NA values in a specific column (e.g. 'S') with 0?
df['S'] = df['S'].fillna(0)
# or in-place (older pandas):
df['S'].fillna(0, inplace=True)
Q7
How do you remove duplicate rows from a DataFrame?
df.drop_duplicates(inplace=True)
# Keep first occurrence by default.
# To keep last: df.drop_duplicates(keep='last', inplace=True)
# To check subset of columns: df.drop_duplicates(subset=['col1','col2'])
Q8
How do you replace values in 'SD0' that are less than or equal to 0 with NaN, using .where()?
import numpy as np
df['SD0'] = df['SD0'].where(df['SD0'] > 0, other=np.nan)
# .where(condition) keeps values where condition is TRUE,
# replaces with `other` where condition is FALSE.

Contrast with .mask() which is the inverse — it replaces where the condition is True.

Q9
After replacing bad values with NaN, how do you forward-fill missing values in 'SD0'?
df['SD0'] = df['SD0'].ffill()
# ffill() = forward fill — propagates the last valid value downward.
# Backward fill: df['SD0'].bfill()
Q10
How do you multiply all values in a column by a scalar (e.g. scale 'SD0' by 100)?
df['SD0'] = df['SD0'] * 100
# Pandas applies scalar operations element-wise automatically.
Q11
How do you convert all column names to Title Case?
df.columns = [c.title() for c in df.columns]
# str.title() capitalises the first letter of each word.
# e.g. 'sd0' -> 'Sd0', 'week_num' -> 'Week_Num'
Q12
How do you sort a DataFrame in descending order by a column called 'Sd0'?
df.sort_values('Sd0', ascending=False, inplace=True)
# ascending=True  → smallest first (default)
# ascending=False → largest first
Q13
How do you calculate percent change of a column and store it in a new column 'PCSDx'?
df['PCSDx'] = df['Sd0'].pct_change() * 100
# pct_change() computes (current - previous) / previous
# Multiply by 100 to express as a percentage.
# The first row will always be NaN (no prior value).
Q14
How do you find the min, mean, and max of a column, along with their index positions?
s = df['S']

# Min
print(s.min(), s.idxmin())

# Max
print(s.max(), s.idxmax())

# Mean (no direct idxmean — find closest value)
mean_idx = (s - s.mean()).abs().idxmin()
print(s.mean(), mean_idx)

idxmin() / idxmax() return the INDEX LABEL of the min/max, not the position number.

Q15
Explain what (s - s.mean()).abs().idxmin() does step by step.
s = pd.Series([1, 5, 9, 3])
# s.mean() = 4.5
# s - s.mean() = [-3.5, 0.5, 4.5, -1.5]  (distance from mean)
# .abs()        = [3.5,  0.5, 4.5,  1.5]  (absolute distances)
# .idxmin()     = 1                         (index of smallest distance)
# So index 1 (value=5) is closest to the mean (4.5)
Q16
What is the difference between .where() and .mask() in pandas?
s = pd.Series([1, -2, 3, -4])

# .where(cond) — KEEP where True, replace where False
s.where(s > 0, other=0)    # [1, 0, 3, 0]

# .mask(cond) — REPLACE where True, keep where False
s.mask(s < 0, other=0)     # [1, 0, 3, 0]
# Both give the same result here — they are inverses of each other.

Q18
How do you plot multiple columns (SD0, SD2, SD1neg) against 'Day' on the same axes using a loop?
fig, ax = plt.subplots(figsize=(9, 5))

for col, marker in zip(['SD0', 'SD2', 'SD1neg'], ['o', 's', '^']):
    ax.plot(df['Day'], df[col], marker=marker, label=col)

ax.set_xlabel('Day')
ax.set_ylabel('Mass (kg)')
ax.set_title('Day vs SD0, SD2, SD1neg')
ax.legend()
plt.tight_layout()

zip() pairs each column name with its marker symbol so you can iterate both together.

Q19
What is the difference between plt.plot() and ax.plot()? When should you use each?
# plt.plot() — implicit, uses the current active axes
plt.plot(x, y)   # quick single plots

# ax.plot() — explicit, targets a specific Axes object
fig, ax = plt.subplots()
ax.plot(x, y)    # preferred when you have multiple subplots
                 # or need fine-grained control

Q21
How do you write a DataFrame to a CSV file without the row index?
df.to_csv('tutorial2.csv', index=False)
# index=False prevents pandas from writing the row numbers as a column.
# index=True (default) would add an extra leftmost column of row numbers.
Q22
How do you check that output files actually exist and print their sizes?
import os

for path in ['tutorial2_plot.png', 'tutorial2.csv']:
    exists = os.path.exists(path)
    size   = os.path.getsize(path) if exists else 'N/A'
    print(f"{path} | exists={exists} | size={size} bytes")
